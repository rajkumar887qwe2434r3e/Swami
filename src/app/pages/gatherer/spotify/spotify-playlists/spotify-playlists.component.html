SPOTIFY Playlists Help

Module : Spotify

Description

This chart shows the playlistis and the number of tracks in each

Execution

This module can be executed by entering a username in the spotify field of the advanced search, being its validation indicator high since it has been entered by the user

It can also be executed if another module has found a spotify username (Fullcontact, Keybase, Searchs, PeopleDataLabs, etc). Its validation indicator will be medium since its discovery is indirect

Finally, and with a low validation indicator, the module can be executed if a username was entered in the "username" field of the advanced search or if a mail was entered in the mail or advanced search. In this case the @domain.ext is cut out and the username is used as input

Source

Spotify API
