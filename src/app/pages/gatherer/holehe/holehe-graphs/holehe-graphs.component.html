Module : Holehe

Description

This graph shows in the network nodes the sites where the mail was successfully found using the forgotten password function

Execution

This module can be executed only by entering an email in the email field of the advanced search or in normal search, being its validation indicator high since it has been entered by the user

Source

The data collected arise from the integration of the holehe project which can be found at

https://github.com/megadose/holehe

holehe allows you to check if the mail is used on different sites like twitter, instagram and will retrieve information on sites with the forgotten password function

Important

Holehe does not alert the target email (https://github.com/megadose/holehe/issues/12)
